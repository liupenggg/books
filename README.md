# 乐优
乐优商城

1. ly-registry：第一步创建eureka微服务

2. ly-gateway：第二步创建网关微服务

3. ly-item：第三步创建商品微服务，先重启网关微服务，再重启商品微服务

4. ly-common：第四步创建通用的工具模块

5. ly-search：搜索微服务

6. ly-page：静态页微服务

7. ly-user：用户微服务

8. ly-sms：短信微服务

9. ly-upload：上传微服务

10. ly-auth：认证微服务

11. ly-cart：购物车微服务

12. ly-order：订单微服务
